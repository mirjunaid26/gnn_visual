GNN Training Visualization

Watch a Graph Neural Network learn to process 4x4 matrices as graphs

About This Visualization

This interactive visualization demonstrates how a Graph Neural Network (GNN) processes 4x4 matrices by treating them as graphs. Each cell in the matrix becomes a node in the graph, with edges connecting adjacent cells.

The network learns to classify matrices based on their structural properties and node values, showing how GNNs can learn from both node features and graph topology.

  • Node color indicates value intensity

Node Embeddings

Visualization of how the GNN transforms node features through message passing. Watch how node representations evolve during training.

  • Each point represents a node

Message Passing

See how information flows between nodes during the GNN's message passing phases. Highlighted edges show active message paths.

  • Arrows show information flow
  • Color intensity shows message strength
  • Updates in real-time during training

Graph Attention

Visualization of attention weights between nodes, showing which connections the GNN considers most important.

  • Darker edges = stronger attention
  • Direction shows information flow
  • Updates each training step
